to run JavaScript code:
1. cd code/javascript
2. node filename.js

to run Python code:
1. cd code/python
2. python filename.py

to run PHP code:
1. cd code/php
2. php ./filename.php

to run GO code:
1. cd code/go/foldername
2. go run filename.go

to run Perl code:
1. cd code/perl
2. perl filename.pl
